# sensor_lab

センサを使って色々遊んでいくリポジトリ。

## ディレクトリ構成

```
sensor_lab/
├── README.md              # センサ一覧インデックス（このファイル）
└── sensors/
    └── {センサ名}/
        ├── README.md      # 型番・スペック・メモ
        └── images/        # センサの画像
```
